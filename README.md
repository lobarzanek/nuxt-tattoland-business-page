# TattoLand - Business web page

## Stack

- Nuxt.js
- Tailwind
- daisyUI

## Live view

https://tattoland.netlify.app/

## Setup

Make sure to install the dependencies:

```bash
# npm
npm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev
```
